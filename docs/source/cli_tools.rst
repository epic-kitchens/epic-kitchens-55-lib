Action segmentation
-------------------

A script is provided to segment frames from a video into a set of
action-segments.


.. code-block:: shell

    $ python -m epic_kitchens.preprocessing.split_segments \
        P03_01 \
        path/to/frames \
        path/to/frame-segments \
        path/to/labels.pkl \
        RGB \
        --fps 60 \
        --frame-format 'frame_%010d.jpg' \
        --of-stride 2 \
        --of-dilation 3
